Was sind BB-Codes überhaupt?
Mit BB-Codes können Sie Ihren, im Forum geschriebenen Text, formatierten. Es sind dafür nur Kürzel notwendig. Eine Übersicht über die Möglichkeiten die Ihnen BB-Codes in diesem Forum bieten, sehen Sie, in der nun folgenden Übersicht.

Übersicht

  • Standard-BB-Codes:
    • [b]Text[/b]
    • [i]Text[/i]
    • [u]Text[/u]
    • [color=#??????]Text[/color]
    • [img]Bildadresse[/img]
    • [url]Adresse[/url] und [url=Adresse]Linkname[/url]
    • [email]Emailadresse[/email]
    • [ot]Text[/ot]
    • [quote]Zitat[/quote] und [quote=Quelle]Zitat[/quote]
    • [list]Listenpunkte[/list] und [list=??]Listenpunkte[/list]
  • Erweiterte BB-Codes:
    • [size=?]Text[/size]
    • [h=?]Text[/h]
    • [align=?]Ausgerichteter Text[/align]
    • [code]Syntax[/code]
    • [note=Erklärung]Wort[/note]
    • [edit]Nachträgliche Anmerkung[/edit] und [edit name=Editor]Nachträgliche Anmerkung[/edit]
    • [hide]unsichtbarer Text[/hide]
    • [sub]Text[/sub] und [sup]Text[/sup]
    • [reader]
    • [table]Text[/table] und [tab]
    • [tt]Text[/tt]
  • Smileys

Standard-BB-Codes

[B]Text[/B]
Hiermit kann ein Text fett dargestellt werden.

Aus [B]Text[/B] wird Text
[I]Text[/I]
Hiermit kann ein Text kursiv dargestellt werden.

Aus [I]Text[/I] wird Text
[U]Text[/U]
Hiermit kann ein Text unterstrichen dargestellt werden.

Aus [U]Text[/U] wird Text
[COLOR=#??????]Text[/COLOR]
Hiermit kann ein Text farbig dargestellt werden.

Aus [COLOR=#ff9900]Text[/COLOR] wird Text
[IMG]Bildadresse[/IMG]
Hiermit kann ein Bild eingefügt werden.

Aus [IMG]img('user'); ?>[/IMG] wird
1. [URL]Adresse[/URL] und 2. [URL=Adresse]Linkname[/URL]
1. Hiermit kann eine URL als klickbarer Link dargestellt werden.
2. Hiermit kann auch eine URL als klickbarer Link dargestellt werden, jedoch kann der Linkname selbst bestimmt werden.

1. Aus [URL]http://www.mamo-net.de[/URL] wird http://www.mamo-net.de/
2. Aus [URL=http://www.mamo-net.de]MaMo Net[/URL] wird MaMo Net
[EMAIL]Emailadresse[/EMAIL]
Hiermit kann eine Email-Adresse als klickbarer Link dargestellt werden. Die Emailadresse wird zum Schutz vor Emailsammlern verschlüsselt!

Aus [EMAIL]user@provider.com[/EMAIL] wird cb_mail("user","provider.com"); ?>
[OT]Text[/OT]
Hiermit kann man Text der nicht zum Thema passt (Off-Topic) markieren.

Aus [OT]Dieser Text passt nicht zum Thema.[/OT] wird
Off-Topic:
Dieser Text passt nicht zum Thema.

Erweiterte BB-Codes

1. [quote]Zitat[/quote] und 2. [quote=Quelle]Zitat[/quote]
1. Hiermit kann ein einfaches Zitat ohne Quellenangabe eingefügt werden.
2. Hiermit kann ein Zitat jedoch mit Quellenangabe eingefügt werden. Die Quellenangabe kann sowohl ein Link als auf ein Name sein.

1. Aus [QUOTE]Dieser Text wird zitiert.[/QUOTE] wird
Zitat:
Dieser Text wird zitiert.

2. Aus [QUOTE=http://www.mamo-net.de]Viscacha ist cool![/QUOTE] wird
Zitat von http://www.mamo-net.de:
Viscacha ist cool!
1. [list]Listenpunkte[/list] und 2. [list=??]Listenpunkte[/list]
1. Hiermit kann eine normale Liste eingefügt werden.
Aus [list][*]Punkt 1 [*]Punkt 2 [/list] wird
  • Punkt 1
  • Punkt 2

2. Hiermit kann man eine erweitere Liste mit anderen Aufzählungszeichen eingefügt werden. Möglichkeiten für Aufzählungszeichen sind "a", "A", "I", "i" und "ol".
Aus [list=ol][*]Punkt 1 [*]Punkt 2 [/list] wird
  1. Punkt 1
  2. Punkt 2
Aus [list=A][*]Punkt 1 [*]Punkt 2 [/list] wird
  1. Punkt 1
  2. Punkt 2
Aus [list=a][*]Punkt 1 [*]Punkt 2 [/list] wird
  1. Punkt 1
  2. Punkt 2
Aus [list=I][*]Punkt 1 [*]Punkt 2 [/list] wird
  1. Punkt 1
  2. Punkt 2
Aus [list=i][*]Punkt 1 [*]Punkt 2 [/list] wird
  1. Punkt 1
  2. Punkt 2
[SIZE=?]Text[/SIZE]
Hiermit kann ein Text vergrößert (large), verkleinert (small) oder erweitert (extended) werden.

Aus [SIZE=small]Text[/SIZE] wird Text
Aus [SIZE=extended]Text[/SIZE] wird Text
Aus [SIZE=large]Text[/SIZE] wird Text
[H=?]Überschrift[/H]
Hiermit können Überschiften erzeugt werden. Mögliche Werte von Überschrift (Groß) bis Unterunterüberschrift (Klein): large, middle, small

Aus [h=large]Überschrift[/h] wird

Überschrift

Aus [h=middle]Unterüberschrift[/h] wird
Unterüberschrift
Aus [h=small]Unterunterüberschrift[/h] wird
Unterunterüberschrift
[ALIGN=?]Ausgerichteter Text[/ALIGN]
Hiermit können Elemente wie Text, Grafiken usw. ausgerichtet werden. Mögliche Werte sind "right", "left", "center" und "justify".

Aus [ALIGN=right]rechtsbündiger Text[/ALIGN] wird
rechtsbündiger Text

Aus [ALIGN=center]zentrierter Text[/ALIGN] wird
zentrierter Text

Aus [ALIGN=left]linksbündiger Text[/ALIGN] wird
linksbündiger Text

Aus [ALIGN=justify]Die Entwicklung des Computers ...[/ALIGN] wird

Die Entwicklung des Computers begann während des 2.Weltkriegs (1939-1945) in Deutschland, England und den USA. In den 30er Jahren speicherten Lochstreifen Daten. 1937 konstruierte Konrad Zuse den ersten Computer, den Z1. Der Z3, der erste programmgesteuerte Rechner, folgte 1941. Zur gleichen Zeit wurden mehrere Spezialrechner (z.B. der Colosseum) für die Entzifferung militärischer Nachrichten entwickelt. Auch in den USA gab es mehrere Entwicklungen. Eine der bedeutendsten war der ENIAC. John von Neumann entwickelte ihn 1945 zum Universalrechner weiter. Diese Weiterentwicklung des ENIACs konnte man mit dem Computer im heutigen Sinne vergleichen. 1948 wurden die ersten Computer mit von-Neumann-Architektur in England vorgestellt (MADM, EDSAC). Die USA bestimmte die weitere Entwicklung.

[CODE]Syntax[/CODE]
Hiermit kann Quelltext eingefügt werden. Einzeiliger Code wird im Fließtext Kastanienrot markiert.

Aus
[CODE]<?php
echo phpinfo();
?>[/CODE]
wird z.B.
Quelltext:
1:
2:
3:
<?php
echo phpinfo();
?>
[NOTE=Erklärung]Wort[/NOTE]
Hiermit kann eine kurze Erklärung zu einer Textstelle (Wort) eingefügt werden.

Aus [note=Abkürzung für Hypertext Markup Language]HTML[/note] wird z.B. HTML.
1. [edit]Nachträgliche Anmerkung[/edit] und 2. [edit name=Editor]Nachträgliche Anmerkung[/edit]
1. Hiermit kann eine nachträgliche Bemerkung im Text merkiert werden. Eingesetzt werden sollte der Tag beim Editieren eines Beitrags.
2. Auch hiermit kann eine nachträgliche Bemerkung im Text merkiert werden, jedoch wird der Name des "Editors" zusätzlich angezeigt wird. Eingesetzt werden sollte der Tag (von Admins, Moderatoren etc.) beim Editieren eines fremden Beitrags.

Beispiel:
Aus [edit]Blablabla[/edit] wird
Nachträgliche Anmerkung des Autors:
Blablabla
[hide]unsichtbarer Text[/hide]
Mit diesem Befehl kann man Text vorerst unsichtbar machen. Es ist gedacht für Bemerkungen für eine spätere Editierung oder Fortsetzung.
1. [sub]Text[/sub] und 2. [sup]Text[/sup]
1. Hiermit kann Text hochgestellt werden.
2. Hiermit kann Text tiefgestellt werden.

1. Aus km[sup]2[/sup] wird zum Beispiel km2
2. Aus H[sub]2[/sub]O wird zum Beispiel H2O
[reader]
Hiermit kann man jeden Nutzer persönlich ansprechen.

Aus "Hi [reader]" wird z.B. "Hi name; ?>".
[table]...[/table] und [tab]
Hiermit kann man Tabellen erstellen. Mit [tab] kann man zusätzlich noch Text einrücken (Tabulator).

Beide Tags zusammen kann man so benutzen, dass man damit einfache Tabellen erstellen kann.
Beispiel:
[table][b]#[tab]Homepage[tab]Traffic[/b]
1.[tab]MaMo Net[tab]500 MB
2.[tab]Smileyarchiv[tab]4 GB
3.[tab]Viscacha[tab]1000 KB [/table]

Ausgabe:
#   Homepage        Traffic
1.  MaMo Net        500 MB
2.  Smileyarchiv    4 GB
3.  Viscacha        1000 KB

Aus [tt]Schreibmaschinenähnlicher Text[/tt] wird Schreibmaschinenähnlicher Text
